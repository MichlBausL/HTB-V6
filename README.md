# HTB-V6 - Haustechnik Bielmeier Website

Next.js Website für IONOS Deploy Now.

## Deployment auf IONOS

1. Dieses Repository zu GitHub pushen
2. In IONOS Deploy Now das Repository verbinden
3. IONOS erkennt automatisch Next.js und baut die Seite

## Lokale Entwicklung

```bash
npm install
npm run dev
```

Die Seite ist dann unter http://localhost:3000 erreichbar.
